Module: authController

Controller de Autenticação da aplicação MVC631. Gerencia o fluxo de login, logout e controle de sessão dos usuários, interceptando as requisições HTTP das rotas de autenticação.

Methods

(static) getLogin(req, res) → {void}

Renderiza a página de login. Redireciona para a home caso o usuário já possua uma sessão ativa, evitando acesso redundante à tela de login. Os cabeçalhos de cache são configurados para impedir que o navegador armazene a página e permita retornar a ela com o botão "Voltar".
Parameters:
Name Type Description
req Request Objeto de Requisição do Express. Verifica `req.session.user` para detectar sessão ativa.
res Response Objeto de Resposta do Express.

(static) login(req, res) → {void}

Processa as credenciais enviadas pelo formulário de login. Valida o e-mail e a senha contra o Model de usuários. Em caso de sucesso, persiste os dados do usuário na sessão e redireciona para a home. Em caso de falha, retorna HTTP 401 e renderiza novamente a view com mensagem de erro.
Parameters:
Name Type Description
req Request Objeto de Requisição do Express. Espera os campos `email` e `senha` no corpo da requisição (`req.body`).
res Response Objeto de Resposta do Express.

(static) logout(req, res) → {void}

Encerra a sessão do usuário autenticado e redireciona para a tela de login. Destrói a sessão do lado do servidor e limpa o cookie `connect.sid` no cliente. Os cabeçalhos de cache são configurados para evitar que páginas autenticadas fiquem acessíveis via histórico do navegador após o logout.
Parameters:
Name Type Description
req Request Objeto de Requisição do Express. Utiliza `req.session.destroy()` para invalidar a sessão ativa.
res Response Objeto de Resposta do Express.
